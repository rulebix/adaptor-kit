# Rulebix Adaptor Kit

Package for creating IDE adapters that generate config.yml files.

## What is this?

This Rulebix package contains rules, skills, and workflows to help you create IDE adapters. The adapter will generate a config.yml file that transforms IDE documentation into Rulebix package format.

## How to Use

1. **Fill IDE Documentation** → Open ide-documentation.md and paste the documentation provided by your IDE
2. **Fill Metadata** → Open metadata.md and fill in adapter information
3. **Generate Adapter** → Run /generate-adapter

## Package Contents

- **4 Rules** - Standards for adapter structure, mapping, expressions, and frontmatter
- **1 Skill** - Complete guide for adapter development with examples
- **1 Workflow** - Auto-generate adapter from IDE documentation
- **2 Placeholders** - Forms for IDE documentation and metadata input
